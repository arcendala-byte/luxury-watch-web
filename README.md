## Using Prisma (PostgreSQL)

This project uses Prisma to access a PostgreSQL database. After setting `DATABASE_URL` in your `.env`, run:

```bash
npm install                # install dependencies including prisma
npx prisma generate        # generate client
npx prisma migrate dev --name init  # create database and apply schema
npx prisma db seed         # populate products (uses prisma/seed.ts)
```

The seed script upserts six luxury watches; modify `prisma/seed.ts` to change data.
